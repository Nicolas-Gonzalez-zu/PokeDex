## Routes/Rutas

> http://localhost:3001/pokemons/pagination In this endpoint you can use offset

> http://localhost:3001/pokemons/pokemon In this endpoint you can use url to spicify a pokemon with the url of the api https://pokeapi.co/

> http://localhost:3001/pokemons/searchPokemon In this endpoint you can search a pokemon by name.

## Try yourself/Pruébelo usted mismo

1. `git clone https://github.com/Nicolas-Gonzalez-zu/pokedex-backend.git`
2. Run `npm install` or `yarn install`
3. Start the dev server using `npm start`
4. Open [http://localhost:3001](http://localhost:3001)
